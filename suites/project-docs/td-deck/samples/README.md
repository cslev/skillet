# TD Deck Samples

Drop files into this directory to calibrate the skill's output:

- **`template.pptx`** — your organization's PowerPoint template. If present, all generated decks are built from this base.
- **`specific_instructions.md`** — slide-by-slide mapping instructions for your template. The skill reads this to know which TD sections go where, slide count limits, and which slides require user input. A starter template is provided — fill it in for your specific `template.pptx`.
- **Any other `.pptx` files** — example TD presentation decks from your organization, used for visual and structural calibration (slide density, layout conventions, figure treatment).

All files in this directory are gitignored (except this README) — safe to include company-specific details.
